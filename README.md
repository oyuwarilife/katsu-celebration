# ポモ活スタンプシート

ポモドーロ活動（朝活・昼活・夜活）の参加記録をスタンプカード形式で残せるWebアプリです。

## リンク

https://oyuwarilife.github.io/katsu-celebration/

## 使い方

1. **アイコン設定**（任意）: 画面上部のカメラアイコンをタップして、自分のアイコン画像をアップロード
2. **時間帯を選ぶ**: 「朝活」「昼活」「夜活」から参加した時間帯をタップ
3. **日付を確認**: 今日の日付が自動入力済み。過去の記録をつけたい場合は日付を変更
4. **「参加した！」をタップ**: お祝い画面が表示され、スタンプカードに肉球スタンプが押される
5. **記録を共有**（任意）: お祝い画面から「Threadsに投稿」や「画像を保存」ができる

## 機能

- カレンダー形式のスタンプカード（月ごとの季節テーマ付き）
- 1日最大3つのスタンプ（朝・昼・夜）
- 同じ日・同じ時間帯の重複登録防止（登録済みは「済」バッジ表示）
- 連続参加日数の表示（2日以上連続で表示）
- お祝いアニメーション（紙吹雪 + 効果音）
- Threads共有 / スタンプカード画像の保存
- スタンプの取り消し（日付をタップ、カスタムダイアログで確認）
- 10個ごとのマイルストーンポップアップ（Googleフォーム誘導）
- PWA対応（ホーム画面に追加してアプリとして使用可能）
- スマホ対応（小さい画面でもレイアウト崩れなし）
- データはブラウザのlocalStorageに保存（アカウント登録不要）

## 技術構成

- HTML / CSS / JavaScript（単一ファイル、フレームワークなし）
- Web Audio API（効果音）
- Canvas API（共有用画像の生成）
- Web App Manifest（PWA対応）
- GitHub Pages でホスティング
